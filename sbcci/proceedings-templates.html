ACM SIG Proceedings Templates

ACM SIG Proceedings Templates

The official ACM Proceedings Format is available as templates in the following formats.

  • Word
  • A CHOICE of LaTeX2e Styles!

Option 1: LaTeX2e - Strict Adherence to SIGS style

Option 2: LaTeX2e - Tighter Alternate style - (may reduce the total number of pages)

Option 1: LaTeX2e - Strict Adherence to SIGS style

Use this set of files to produce articles with NO permission block, copyright line or page numbering.
This style strictly adheres to the SIGS (Pubs-board-endorsed) style.

  • ACM LaTeX2e Sample File (APRIL 2009 PDF)
  • ACM LaTeX2e Style File V3.2SP (APRIL 2009 CLS)
  • ACM LaTeX2e Sample File V3.1 (APRIL 2009 LaTeX)
  • ACM LaTeX2e Sample BIB File
  • Graphic #1 (EPS)
  • Graphic #2 (EPS)
  • Graphic #3 (PS)
  • ACM Author Guidelines (LaTeX2e) Microsoft Word Document V2.2
  • ACM Author Guidelines (LaTeX2e) HTML Document V2.2

Option 2: LaTeX2e - Tighter Alternate style - (may reduce the total number of pages)

Use this set of files to produce articles WITH the permission block, conference location, copyright line, etc.
This style essentially conforms to the style yet produces a tighter-looking format which invariably means fewer pages.

  • Alternate ACM LaTeX2e Sample File (APRIL 2009 PDF)
  • Alternate ACM LaTeX2e Style File V2.4 (APRIL 2009 CLS)
  • Alternate ACM LaTeX2e Sample File V1.9 (APRIL 2009 LaTeX)
  • ACM LaTeX2e Sample BIB File
  • Graphic #1 (EPS)
  • Graphic #2 (EPS)
  • Graphic #3 (PS)
  • Alternate ACM Author Guidelines (LaTeX2e) Microsoft Word Document V1.1
  • Alternate ACM Author Guidelines (LaTeX2e) HTML Document V1.1

If you have LaTeX-specific questions please consider checking out the SIG FAQ FILE first.
All other questions can go to Adrienne Griscti at griscti@acm.org
